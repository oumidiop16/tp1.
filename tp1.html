    Les jeux de caractères (Charset)

    Un jeu de caractères (encodage de caractères) est une façon de représenter chaque caractère (et donc un texte) dans cette base binaire.
    Pour afficher une page HTML correctement, un navigateur Web doit savoir le jeu de caractères à utiliser.

    Les jeux de caractères usuelles :

  6. ASCII a été la première norme d’encodage de caractères. Il définit 128 différents caractères alphanumériques qui pourraient être utilisés sur Internet: les chiffres (0-9), les lettres anglaises (AZ) et certains caractères spéciaux comme ! $ + - ( ) @ < et >.

  7. ANSI (Windows 1252) était le jeu de caractères d'origine de Windows. Il supporte 256 codes de caractères différents.

  8. ANSI (Windows 1252) était le jeu de caractères d'origine de Windows. Il supporte 256 codes de caractères différents.

  9. ISO-8859-1 est le jeu de caractères par défaut pour HTML 4. Il prend en charge également 256 codes de caractères différents.

  10. ANSI et ISO-8859-1 étant si limitées, HTML 4 prend en charge également UTF-8.

  11. UTF-8 (Unicode) couvre presque tous les caractères et les symboles dans le monde.

  12. L’encodage de caractère par défaut pour HTML5 est UTF-8.

    Les entités et symboles
    • Certains caractères sont réservés en HTML. Il n'est pas possible d'utiliser le signe inférieur (<) ou supérieur (>) dans votre texte, parce que le navigateur les prendra pour des balises.

    • Pour afficher réellement les caractères réservés, nous devons utiliser les entités de caractères dans le code source HTML.

    • Syntaxe d’une entité: &entity_name; ou &#entity_number;

    • Exemple : Pour afficher le signe inférieur (<) il faut écrire: "& lt"; ou "& #60"

    • Avantage et inconvénient : L'avantage d'utiliser un nom d'entité, au lieu d'un nombre, c'est que le nom est facile à retenir. Mais l'inconvénient est que les navigateurs ne supportent pas tous les noms d'entité.

    TABLEAUX ET LISTES

    Les tableaux

    Définition

    • Une table HTML est définie par la balise < table>.

    • Chaque ligne de la table est définie par la balise < tr>.

    • Chaque en-tête est définie avec la balise < th> et est par défaut en gras et est centré.

    • Chaque cellule est définie avec la balise < td>.

    Remarque:

    • Les éléments < td> sont les conteneurs de données de la table

    • Ils peuvent contenir toutes sortes d'éléments HTML: du texte, des images, des listes, d'autres tables, etc.

    • colspan : Indique que la cellule courante s'étend sur plusieurs colonnes.

    • colspan : Indique que la cellule courante s'étend sur plusieurs ligne.

    Les liens

    Définition :

    <"ul style="list-style-type:circle">
  13. Les liens sont présents dans presque toutes les pages Web. Ils permettent aux utilisateurs de se déplacer de page en page.
  14. Un lien (hyperlien) est un mot, un groupe de mots ou une image qui peut être cliquer pour passer à un autre document ou une
    autre section dans le document courant.
  15. Lorsque le curseur est déplacé sur un lien dans une page Web, la flèche se transforme en une petite main.
  16. Les liens sont spécifiés en utilisant la balise "".

    17. Syntaxe :

  17. Le code HTML pour créer un lien est simple.Elle ressemble à ceci: "<"a href="url">Texte du lien"
    L'attribut hrefindique la destination du lien.
  18. Remarque: Le texte du lien peut être du texte, une image ou tout autre élément HTML.
  19. Exemple:
    "<"a href="http://www.ugb.sn">Visiter UGB"

    Couleur des liens :

    Par défaut, un lien apparaîtra comme dans tous les navigateurs:
    • Un lien non visité est souligné et de couleur bleu
    • Un lien visité est souligné et de couleur violet
    • Un lien actif est souligné et de couleur rouge

    L'attribut target :

    L'attribut target spécifie où s'ouvrira le document lié. Il peut avoir l'une des valeurs suivantes :
    • _blank- ouvre le document lié dans une nouvelle fenêtre ou onglet
    • _self - ouvrir le document lié dans la même fenêtre/onglet que le document contenant le lien (c’est la valeur par défaut)
    • _parent- Ouvre le document lié dans le cadre parent
    • _top- Ouvre le document lié dans le corps entier de la fenêtre
    • nomFrame- Ouvre le document lié dans le cadre nommé nomFrame
    • Exemple : <"a href="http://www.ugb.sn" target="_blank">Visiter UGB

    Signet HTML :

    • Un signet est utilisé pour permettre aux lecteurs de passer à des parties spécifiques d'une page Web.
    • Ils sont surtout utiles lorsque la page Web est très longue.
    • Une fois le signet créé, on peut créer un lien vers lui.

    Les iframes :

    Définition :

    • Une iframe est utilisée pour afficher une page Web dans une page web.La balise iframe est suivie d'un src="URL" "URL" pointe vers l'emplacement de la page
    • Les attributs heightet widthspécifient la hauteur et la largeur de l'iframe.
